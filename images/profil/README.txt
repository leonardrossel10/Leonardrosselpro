Placez ici votre photo de profil et, si souhaité, la vidéo de présentation.

Images :
- Nom recommandé pour la photo : `profile.jpg` (JPEG/PNG). La page accepte aussi `IMG_5545.JPG` et `IMG_5548.JPG`.

Vidéo :
- Créez le dossier `images/profil/videos/` (déjà créé) et déposez-y votre vidéo principale nommée `intro.mp4` :
	`images/profil/videos/intro.mp4`
- Formats recommandés : `mp4` (H.264) pour compatibilité maximale.
- Si tu préfères héberger la vidéo sur YouTube, remplace la valeur `data-video` dans les triggers de `gallery.html` par l'URL YouTube complète (ex: `https://youtu.be/ID`).

Exemples :
- Local : `images/profil/videos/intro.mp4`
- YouTube : `data-video="https://youtu.be/abcd1234"`

La page `gallery.html` utilise maintenant par défaut `images/profil/videos/intro.mp4`.
